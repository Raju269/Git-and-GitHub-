### Common Problems Faced While Learning Git & GitHub Problems Faced by Students

- Confusion between Git (tool) and GitHub (platform)

- Fear of using the terminal/command line

- Difficulty understanding commit, push, and pull

- Mistakes while pushing code to GitHub

- Not knowing how to write a proper commit message

- Trouble understanding repositories and branches

- Unsure how Git is used in real projects

- Forgetting commands due to lack of practice

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

### Problems Faced by Employees

- Difficulty understanding existing project Git workflows

- Confusion while working with branches and merges

- Fear of breaking production code

- Merge conflicts and how to resolve them

- Understanding pull requests and code reviews

- Synchronizing work with team members

- Managing frequent updates from the remote repository

- Maintaining clean commit history


-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


### Why These Problems Happen

- Learning Git without hands-on practice

- Jumping directly to advanced topics

- Not following a step-by-step learning path

- Lack of real-world examples

- No proper beginner-friendly documentation
